Composable Type Systems

Introduction

One of the goals of OftLisp is to be able to make a complete Lisp system that runs on bare-metal (i.e. as an operating system). To that end, it makes a lot of sense to be able to have multiple sublanguages. This is how Lisp has been written traditionally, as a language which one modifies to make their own specific DSL. However, one would also wish to have a typed language when developing a large software system.
